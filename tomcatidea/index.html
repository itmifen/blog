    Tomcat多站点部署

    Tomcat的目录机构主要包括:
    | 目录或文件 | 说明 |
    | —————– |:————-:|
    | bin | 运行命令的目录 |
    | conf | 配置文件目录 |
    | lib | 自带的库文件 |
    | logs | 日志目录 |
    | webapps | 默认站点 |

    进行多站点部署主要是修改conf目录中的server.xml文件，此文件是Tomcat的配置文件。
